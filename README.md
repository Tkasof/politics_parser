# Парсер политических партий России

## 1. Что делает программа и какие данные извлекает

Программа извлекает данные о политических партиях России из HTML-страницы Министерства юстиции:

- **Наименования политических партий** (полные официальные названия)
- **Ссылки на документы** партий (уставы, регистрационные сведения)
- **Обрабатывает отсутствующие ссылки** - устанавливает значение `null` для партий без документов
- **Преобразует относительные ссылки** в абсолютные URL-адреса
- **Структурирует данные** в формате JSON для дальнейшего использования

## 2. Откуда берётся HTML и как его подготовить

**Источник данных:** Официальный сайт Министерства юстиции РФ  
**URL:** https://minjust.gov.ru/ru/pages/politicheskie-partii/

**Подготовка HTML-файла:**

1. Откройте указанный URL в веб-браузере
2. Дождитесь полной загрузки страницы
3. Сохраните страницу на компьютер:
   - Нажмите `Cmd + S` (Mac) или `Ctrl + S` (Windows)
   - Выберите формат "Веб-страница, полностью" или "HTML-код веб-страницы"
   - Сохраните файл под именем `parties_page.html`
4. Поместите сохраненный файл в папку с программой `parse_parties.py`

## 3. Как установить зависимости

Программа требует установки библиотеки BeautifulSoup4. Установка осуществляется через файл `requirements.txt`:

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

## 4. Как запустить программу и просмотреть результат

**Запуск программы:**

```bash
python parse_parties.py
```

**Результаты работы:**

- В консоли отобразится пронумерованный список всех политических партий с ссылками на их документы
- В файле parties.json сохраняются структурированные данные в формате:

```bash
[
  {
    "name": "Полное наименование партии",
    "doc_url": "https://полная-ссылка-на-документ"
  }
]
```

- Для партий без документов в поле doc_url будет указано значение null
