# Парсер политических партий России

Программа для парсинга списка политических партий с сайта Министерства юстиции РФ.

## Что делает программа

- Извлекает список политических партий России
- Для каждой партии находит название и ссылки на документы (Устав, регистрационные данные)
- Преобразует относительные ссылки в абсолютные
- Сохраняет данные в структурированном JSON-формате

## Установка

```bash
git clone https://github.com/Tkasof/politics_parser.git
cd politics_parser
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

## Запуск 
(сохранив страницу как parties_page.html)
```bash
python parse_parties.py
```
